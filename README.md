# Bachelor-s-thesis

online E-shop 

Tento projekt predstavuje jednoduchý e-shop implementovaný ako súčasť bakalárskej práce. Aplikácia je vytvorená pomocou technológií PHP, JavaScript a CSS a demonštruje základné princípy vývoja dynamických webových aplikácií.


## Použité technológie:
- PHP (backend logika)
- JavaScript (interaktivita na strane klienta)
- HTML5
- CSS3
- MySQL (databáza)
- Apache (lokálny server, XAMPP)

- ## Funkcionality:
- Zobrazenie zoznamu produktov
- Detail produktu
- Pridávanie produktov do košíka
- Odstraňovanie položiek z košíka
- Správa košíka pomocou PHP session
- Simulácia objednávky
- Práca s databázou (CRUD operácie)
